How to run the project-

1. cd Backend

2. create a virtual environment-
   "python3 -m venv .venv"

3. Run this command to install python dependencies:-
    "pip install -r requirements.txt"

    Now Run the backend server:
    "flask --app main run"

4. "cd frontend" and run the following command:
    npm install (npm package manager required)

    Now run frontend-
      "npm start"

Dashboard should be visible in your browser.
